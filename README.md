EN
Camping system with ox_lib, ox_target, and ox_inventory.
You can set up a tent with an enter/exit option, a chair with a sit option, a table, a cooler with a second inventory function, and a campfire where you can grill meat to eat.
For food, you add your inventory items in the config to be able to grill them.
